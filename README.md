# Rocketseat Go

Projeto de estudo da linguagem Go pela Rocketseat.

## Sobre

Este repositório contém exemplos e exercícios de programação em Go, desenvolvidos como parte do curso da Rocketseat.

## Requisitos

- Go 1.19 ou superior

## Como usar

Clone o repositório:

```bash
git clone https://github.com/jakunzler/rocketseat-go.git
cd rocketseat-go
```

Execute os exemplos:

```bash
go run main.go
```
